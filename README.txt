Il s'agit d'un projet de fin d'année réalisé par Davigny Théo, Laurain Amandine et Marinho Mattéo.

Le projet a pour but d'avoir un réseau social dans lequel les utilisateurs peuvent poser des questions et d'autres peuvent y répondre.

Les technologies utilisées sont :
- HTML, CSS, Bootstrap, PHP v.7.0.3, MySQL

Le rôle de chaque membre de l’équipe : 
Théo D : page d'accueil, gestion de la connexion/inscription des utilisateurs et la gestion des questions/réponses en back end
Matteo M : page de connexion/inscription, page d'accueil
Amandine L : Base de données, page d’accueil et la page des questions/réponses aux questions.

La durée du projet : 09 mars 2021 jusqu'au 30 avril 2021.

Guide d'installation : 

- Installez une plateforme de développement (ex: Uwamp)
- Déposez les fichiers du projet dans le répertoire de travail (ex: /www/ sur Uwamp)
- Accédez à phpmyadmin, créez une base de donnée au nom de "livequestion" et importez le fichier .sql dans celle-ci.
